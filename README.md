

# 💳 Credit Card Approval Predictor

## 📌 Project Overview

This project predicts whether a customer’s credit card application will be **approved or rejected** using **Machine Learning 
(Logistic Regression)**. It helps financial institutions automate decision-making and reduce risk.

---

## 🎯 Objective

* Build a classification model to predict approval status
* Analyze customer financial and personal data
* Improve accuracy of loan/credit approval decisions

---

## 📊 Dataset Features

Typical features used:

* Age
* Income
* Employment Status
* Credit Score
* Existing Loans
* Debt-to-Income Ratio

---

## ⚙️ Technologies Used

* Python 🐍
* Pandas & NumPy
* Matplotlib & Seaborn
* Scikit-learn

---

## 🔄 Project Workflow

1. Data Collection

3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Model Building (Logistic Regression)
6. Model Evaluation
7. Prediction

---

## 🧠 Model Used

* Logistic Regression (Supervised Learning)

---

## 📈 Evaluation Metrics

* Accuracy
* Precision
* Recall
* Confusion Matrix

---



## 📌 Results

* The model successfully predicts credit card approvals
* Helps in faster and more accurate decision-making
